Synthetic & Hybrid Sound Commands

OctaMED 4 Professional — programmable waveform & volume sequences

A synthetic sound is built from raw waveforms driven by two small programs that run on every timing pulse: the Waveform Sequence List (which controls pitch, waveform selection, arpeggio and vibrato) and the Volume Sequence List (which controls volume and envelopes). A hybrid sound pairs an ordinary sample with the same programmable lists, so a recorded sound can be animated by the synth engine.

Each list is a numbered set of lines. A line is either a value (a bare number — a volume or waveform), or a three-letter command optionally followed by an argument on the next line. Values are hexadecimal. The tables below are the full command set; the sections that follow explain each one with the original examples.

Volume Sequence List — commands in detail
Set volumecmd: — · arg: volume · key: (direct number)

The default command — it has no keyword. It sets the absolute volume of the synthetic sound, range 00–40. Relative track volumes from the Volume panel cannot be used with synthetic sounds.

00 30 Volume = $30 01 10 Volume = $10
End sequencecmd: END · arg: — · key: —

Terminates the Volume Sequence List and is always present. You cannot insert commands below it.

Change Down (volume)cmd: CHD · arg: speed · key: D

Sets the speed at which the volume decreases on each timing pulse. The slide starts after the command. To stop it, issue the command again with speed 00.

00 CHD Change Down 01 05 speed = 5 ⋮ 10 CHD Change Down 11 00 speed = 0 → stop sliding
Change Up (volume)cmd: CHU · arg: speed · key: U

As CHD, but slides the volume up.

Waitcmd: WAI · arg: pulses · key: W

Pauses for the specified number of timing pulses.

03 WAI Wait 04 10 wait 16 ($10) pulses
Jumpcmd: JMP · arg: line # · key: J

Goes to another position in the Volume Sequence List.

05 JMP Jump 06 0A jump to line 0A
Jump Waveform Sequencecmd: JWS · arg: line # · key: Shift-J

Causes a jump in the Waveform Sequence — for example to trigger a pitch change at the end of the Volume Sequence List. It does not jump within the Volume Sequence.

04 JWS Jump Waveform Sequence 05 0F jump to waveform sequence 0F
Haltcmd: HLT · arg: — · key: H

Same effect as END, but it can sit in the middle of a sequence list (the code below it can still be reached with JMP).

03 HLT Halt 04 0… other code, reachable via JMP
Speedcmd: SPD · arg: speed · key: S

Sets the execution speed.

0A SPD Speed 0B 01 speed = 1 (fastest)
V2.00+ The following commands require OctaMED V2.00 (MED V3.20) or later.
One-shot Envelopecmd: EN1 · arg: envelope waveform # · key: E

Sets the shape of the envelope. When the end of the envelope is reached, nothing further occurs. The envelope waveform must always be 128 bytes long; execution begins on the next interrupt, so the volume here is initialised to 40.

02 40 03 EN1 Set Envelope 04 05 use waveform 05 as envelope
Looping Envelopecmd: EN2 · arg: envelope waveform # · key: Shift-E

As EN1, except that on reaching the end the envelope loops back to the beginning.

Waveform Sequence List — commands in detail
Set Waveformcmd: — · arg: waveform # · key: (direct number)

Selects the waveform number (from 00). After this instruction execution stops until the next timing pulse. Do not use waveform numbers higher than the last waveform.

00 00 waveform 00 01 01 waveform 01
End sequencecmd: END · arg: — · key: —

Terminates the Waveform Sequence List and is always present. You cannot insert commands below it.

Change Down (pitch)cmd: CHD · arg: speed · key: D

Sets the speed for sliding the pitch down. The slide starts after the command and stops when the speed is set to zero.

00 CHD Change Down 01 03 speed = 3
Change Up (pitch)cmd: CHU · arg: speed · key: U

As CHD, but slides the pitch up.

Waitcmd: WAI · arg: pulses · key: W

Pauses for the number of pulses specified.

03 WAI Wait 04 02 wait 2 pulses
Jumpcmd: JMP · arg: line # · key: J

Jumps to another position in the Waveform Sequence List.

05 JMP Jump 06 0A go to line 0A
Jump Volume Sequencecmd: JVS · arg: line # · key: Shift-J

Causes a jump in the Volume Sequence — for example to trigger volume changes after waveform events.

09 JVS Jump 0A 00 go to volume sequence 00
Haltcmd: HLT · arg: — · key: H

Same effect as END, but it can sit in the middle of the sequence list.

03 HLT Halt 04 0… other code
Speedcmd: SPD · arg: speed · key: S

Sets the execution speed in pulses.

0A SPD Set Speed 0B 01 speed = 1 (fastest)
Arpeggio Definitioncmd: ARP · arg: offset · key: A

Begins an arpeggio definition. The values that follow are the arpeggio offsets from the root note; the definition ends with ARE. The arpeggio starts once the sequence is defined.

03 ARP Start Arpeggio 04 00 offset 05 04 offset 06 07 offset 08 ARE End Definition
Arpeggio Endcmd: ARE · arg: — · key: E

Ends an arpeggio definition (see above).

Vibrato Depthcmd: VBD · arg: depth · key: V

Sets the vibrato depth (00–7F). Values below 0F suit ordinary vibrato; higher values are mainly for special effects.

02 VBD Set Vibrato Depth 03 04 depth = 4
Vibrato Speedcmd: VBS · arg: speed · key: Shift-V

Sets the vibrato speed (00–7F). Both speed and depth must be non-zero for vibrato to work.

02 VBD Set Vibrato Depth 03 04 depth = 4 04 VBS Set Vibrato Speed 05 30 speed = 30
Reset Pitchcmd: RES · arg: — · key: R

Resets the note to its initial pitch.

V2.00+ The following command requires OctaMED V2.00 (MED V3.20) or later.
Vibrato Waveformcmd: VWF · arg: waveform # · key: Shift-W

Sets the vibrato waveform, which should always be 32 bytes long. It is played reversed (use the REV button to reverse it). A sine wave is used by default.

00 VBD Set Vibrato Depth 01 06 depth = 6 02 VBS Set Vibrato Speed 03 40 speed = 40 04 VWF Set Vibrato Waveform 05 04 use waveform 04
Synthetic sound commands in songs

If used as a MIDI command in a song, E controls MIDI pan. With synthetic sounds, however, E triggers a jump in the Waveform Sequence List. For example, to drop the pitch of a sound at a chosen point, the Waveform Sequence might read:

00 VBS Set Vibrato Speed 01 40 speed = 40 02 VBD Set Vibrato Depth 03 06 depth = 6 04 00 Play Waveform 00 05 HLT Halt 06 CHD pitch-changing entry point 07 02 speed = 2 08 END End

You could then compose a track like this:

011 | C-2 30000 the previous synthetic sound 013 | --- 00000 014 | --- 00000 015 | --- 00E06 jump to position 06 → pitch slides down 016 | --- 00000 017 | …
Hold and Decay in synthetic sounds

Hold and Decay work well with synthetic sounds. The decay value here is a jump address in the Volume Sequence List: when decay starts, execution jumps to that line. It can also reach the Waveform Sequence List via the JWS command.

00 40 Initialise volume 01 HLT Halt 02 CHD decay handling (entry point) → cause decay 03 03 speed = 3 04 END End

The decay value for this synthetic sound should be 2.
