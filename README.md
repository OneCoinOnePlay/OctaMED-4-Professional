# OctaMED 4 Professional — A Concise Guide

A concise guide and complete keyboard-shortcut reference for **OctaMED 4 Professional**,
the legendary music tracker for the Commodore Amiga, written by **Teijo Kinnunen**.

Open [`index.html`](index.html) in any browser to read it.

## What is OctaMED?

**OctaMED** grew out of *MED* — the *Music EDitor* — and takes its name from *octa* (eight):
it mixes eight voices in software on a machine whose Paula sound chip offered only four
hardware channels. Version 4 Professional, distributed in the UK by **RBF Software**
(Ray Burt-Frost), became one of the defining tools of the early-1990s Amiga scene.

Songs are built from **blocks** of **tracks**, chained into a **play sequence**, and driven
almost entirely from the keyboard — which is why a good shortcut reference matters.

## The basics in one minute

| Term | Meaning |
| --- | --- |
| **Sample / instrument** | A digitised sound in the instrument list (up to 63 per song), played from the keyboard across selectable octaves. |
| **Track** | A single column of notes — one voice. Up to eight play at once. |
| **Block** | A pattern: a grid of tracks and lines (commonly 64). You edit one at a time. |
| **Play sequence** | The running order of blocks that forms the whole song. |
| **Command** | The hex effect column beside each note (volume, portamento, tempo, …). `Alt`+number adds 10 to command values. |

**Workflow:** load instruments → enter notes into a block → chain blocks in the play
sequence → press `Amiga-p` to play the song.
